## To run this 
```bash
npx shivadev
```

### Local installation

```bash
git clone git@github.com:ShivaBhattacharjee/shiva-npm.git
```
```bash
cd shiva-npm
```
```bash
npm i
```
### Build 
```bash
npm run build
```
### Start
```bash
npm start
```
